Features

Paste without the mess.
Every time.

🔨

One-Toggle Simplicity

Click the menu bar icon. Flip the switch. That's it. When it's on, every paste is plain text. When it's off, normal paste.

🌐

System-Wide

Works in every app — Mail, Slack, Notion, Pages, browsers, terminals. Wherever you paste, formatting is gone.

🗑

Strips RTF & HTML

Rich text, HTML markup, font styles, colors, sizes — all stripped down to clean, pure plain text on every paste.

📄

Preserves Plain Text

Already plain text? It stays exactly as-is. No mangling, no encoding issues. Just clean text in, clean text out.

📊

Stats Tracking

See how many times formatting has been stripped — lifetime and per session. A small joy every time you paste clean.

🚀

Launch at Login

Enable it once and forget about it. Plain Paste starts silently with your Mac, lives quietly in the menu bar.

How It Works

Stupid simple.
By design.

1

Toggle on

Click the clipboard icon in your menu bar and flip the switch. Plain Paste is now active and monitoring your clipboard.

2

Copy anything

Copy from anywhere — emails with bold text, web pages with custom fonts, documents with colored headings. Doesn't matter.

3

Paste plain text

Hit Cmd+V like you normally would. The formatting is gone. Just clean, plain text lands in your document. Every single time.
